
# Student Performace Factors Data Analysis in Python

This project explores the relationship between several student educational factors and their effect on student test scores. This is an attempt to find a relationship between two or more factors that increase a student's performance on exams and whether there are other factors that predict a decrease in exam scores.

## Usage/Examples

To use this project, do the following:

    2. Clone this project

    3. Use requirements.txt to set up dependencies

    4. Data will now be available for you to use
